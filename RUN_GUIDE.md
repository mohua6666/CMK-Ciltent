# MK PVP Client - 手机端运行指南

## 📱 在手机上运行 MK PVP Client

恭喜你！MK PVP Client项目已完成开发。本文将详细指导你如何在 Android 手机上运行和测试这个应用。

---

## 第一部分：环境准备

### 1.1 必要工具

| 工具 | 版本要求 | 用途 |
|------|----------|------|
| **Android Studio** | 2022.1 或更高 | 官方 Android 开发 IDE |
| **JDK** | 17 或更高 | Java 编译环境 |
| **Android 手机** | Android 5.0+ | 测试设备 |
| **数据线** | USB 数据线 | 连接手机和电脑 |

### 1.2 下载安装 Android Studio

1. 访问 [Android Studio 官网](https://developer.android.com/studio)
2. 点击 "Download Android Studio"
3. 下载完成后运行安装程序
4. 按照向导完成安装（推荐使用默认设置）

### 1.3 配置手机开发者选项

1. 打开手机 **设置**
2. 进入 **关于手机**
3. 连续点击 **版本号** 7 次，开启开发者模式
4. 返回设置，进入 **系统** → **高级** → **开发者选项**
5. 开启 **USB 调试**

## 第二部分：项目导入与构建

### 2.1 导入项目到 Android Studio

1. 打开 Android Studio
2. 点击 **File** → **Open**
3. 导航到项目文件夹，选择 `settings.gradle` 文件
4. 点击 **OK** 打开项目
5. 等待 Gradle 同步完成（约 3-5 分钟）

### 2.2 配置 Gradle（如果提示）

如果出现 Gradle 版本不匹配的提示：

1. 点击 **File** → **Project Structure**
2. 在 **Project** 选项卡中设置：
   - Gradle Version: `8.2`
   - Android Gradle Plugin: `8.2.0`
3. 点击 **Apply** → **OK**

### 2.3 构建项目

**方式一：构建 Debug APK（推荐用于测试）**

1. 在 Android Studio 右侧找到 **Gradle** 面板
2. 展开 **app** → **build**
3. 双击 **assembleDebug**
4. 等待构建完成

**方式二：通过菜单构建**

1. 点击 **Build** → **Build Bundle(s) / APK(s)**
2. 选择 **Build APK(s)**
3. 等待构建完成

### 2.4 找到生成的 APK

构建成功后，APK 文件位置：

```
项目根目录/app/build/outputs/apk/debug/app-debug.apk
```

## 第三部分：安装到手机

### 3.1 通过 USB 安装

1. 使用 USB 数据线连接手机和电脑
2. 手机上选择 **文件传输 (MTP)** 模式
3. 在 Android Studio 中点击 **Run** → **Run 'app'**
4. 选择你的手机设备
5. 应用将自动安装并启动

### 3.2 手动安装 APK

1. 将 `app-debug.apk` 复制到手机存储
2. 在手机上打开文件管理器
3. 找到 APK 文件，点击安装
4. 如果提示"未知来源"，前往 **设置** → **安全** → 开启 **未知来源**

### 3.3 安装后的首次设置

1. 打开 MK PVP Client
2. 首次使用需要授予权限：
   - 存储权限（访问游戏文件）
   - 网络权限（下载版本）
3. 应用会自动创建必要的文件夹

## 第四部分：核心功能使用指南

### 4.1 账号管理

**添加账号：**
1. 点击左上角头像进入账号管理
2. 点击右下角 **+** 按钮
3. 选择在线/离线模式
4. 输入用户名（在线模式需输入密码）
5. 点击登录添加

**选择账号：**
- 点击账号卡片即可选择为当前账号

### 4.2 版本管理

**下载游戏版本：**
1. 进入 **版本管理** 页面
2. 点击右下角 **+** 按钮
3. 从列表中选择要下载的版本
4. 等待下载完成

**选择游戏版本：**
1. 返回主页面
2. 在版本列表中点击要使用的版本
3. 版本会高亮显示为当前选择

### 4.3 MOD 管理

**安装 MOD：**
1. 进入 **MOD管理** 页面
2. 点击右下角 **+** 按钮
3. 从文件管理器选择 `.jar` 格式的 MOD 文件
4. MOD 将自动安装到 mods 文件夹

**管理 MOD：**
- 点击 MOD 可查看详情
- 使用复选框启用/禁用 MOD
- 长按可删除 MOD

### 4.4 材质包管理

**安装材质包：**
1. 进入 **材质包** 页面
2. 点击右下角 **+** 按钮
3. 选择 `.zip` 格式的材质包
4. 材质包将自动安装

### 4.5 存档管理

**备份存档：**
1. 进入 **存档管理** 页面
2. 点击存档右侧的备份图标
3. 或点击右下角按钮一次性备份所有存档

### 4.6 皮肤管理

**导入皮肤：**
1. 进入 **皮肤管理** 页面
2. 点击右下角 **+** 按钮
3. 选择 PNG 格式的皮肤图片
4. 皮肤将自动导入

### 4.7 PVP客户端下载

**下载PVP客户端：**
1. 进入 **PVP客户端** 页面
2. 选择想要的客户端（Moon Client、Lion Client等）
3. 点击下载按钮
4. 等待下载完成

### 4.8 按键映射

**自定义按键布局：**
1. 进入 **按键映射** 页面
2. 长按任意按键可编辑：
   - 按键名称
   - X/Y 坐标位置
   - 按键宽度和高度

**保存布局：**
1. 点击 **保存** 按钮
2. 输入布局名称
3. 点击确定保存

## 第七部分：常见问题

### Q1: 构建失败怎么办？

**解决方法：**
1. 检查 JDK 版本是否为 17+
2. 点击 **File** → **Invalidate Caches / Restart**
3. 删除 `.gradle` 和 `build` 文件夹
4. 重新构建

### Q2: APK 无法安装？

**解决方法：**
1. 检查手机 Android 版本是否 ≥ 5.0
2. 确保已开启"未知来源"权限
3. 检查存储空间是否充足

### Q3: 游戏无法启动？

**解决方法：**
1. 确认已下载游戏版本
2. 检查内存分配是否足够
3. 尝试切换渲染器

### Q4: MOD 不生效？

**解决方法：**
1. 确认 MOD 与游戏版本兼容
2. 检查 MOD 是否启用
3. 部分 MOD 需要特定的 ModLoader（Forge/Fabric）
