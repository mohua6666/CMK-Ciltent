# MK PVP Client

一个专门为 Minecraft PVP 玩家设计的 Android 启动器。

## 作者

**Mohua**

## 功能特性

- ✅ 账号管理 - 支持在线/离线账号
- ✅ 皮肤管理 - 自定义皮肤导入
- ✅ PVP 客户端下载 - 一键下载 Moon Client、Lion Client 等
- ✅ 版本管理 - 下载和管理游戏版本
- ✅ MOD 管理 - 安装和管理 MOD
- ✅ 材质包管理 - 导入材质包
- ✅ 存档管理 - 备份和恢复存档
- ✅ 按键映射 - 自定义虚拟按键
- ✅ PVP 设置 - FPS/延迟显示、攻击距离、点击速度等
- 🎨 暗色 UI 主题 - 红黑配色

## 技术栈

- 语言: Java
- 最低 SDK: 21 (Android 5.0)
- 目标 SDK: 34
- 构建工具: Gradle 8.2
